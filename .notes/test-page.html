Test Article: Understanding Browser Extensions

This is a test article to verify the Gloriosa extension works correctly. The extension should extract this main content and convert it to Markdown format.

Key Features

The extension provides several important features:

  • Content extraction using Readability.js
  • HTML to Markdown conversion using Turndown.js
  • Automatic clipboard copying
  • User feedback with badges

How It Works

When you click the browser action button, the extension:

  1. Shows a loading indicator
  2. Injects the content script into the active tab
  3. Extracts the main content using Readability
  4. Converts the HTML to Markdown
  5. Copies the result to your clipboard
  6. Shows a success or error badge

Code Example

function extractContent() {
	const reader = new Readability(document);
	return reader.parse();
}
